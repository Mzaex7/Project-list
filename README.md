# Max Zeitler | Portfolio Website

A personal portfolio website showcasing projects in Data Science, AI, and Software Engineering.

## Features

- **Responsive Design**: Adapts to all screen sizes.
- **Dynamic Project Loading**: Projects are loaded from a central JSON file.
- **Project Detail Pages**: Individual pages for each project with highlights and links.
- **Dark Mode Aesthetic**: Modern dark theme with accent colors and glow effects.

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (ES6 Modules)
- Font Awesome (for icons)

## Structure

- `index.html`: Home page with selected projects and experience.
- `projects.html`: Full list of projects in a horizontal layout.
- `project-detail.html`: Template for individual project details.
- `js/projects.js`: Data source for all projects.
- `js/main.js`: Core logic for rendering projects and handling UI interactions.
- `css/style.css`: Global styles and layout definitions.

## Live Demo

Run locally with:
```bash
python3 -m http.server
```
Then visit `http://localhost:8000`.
